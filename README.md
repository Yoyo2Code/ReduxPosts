# React Redux Starter Kit

Contains extra to make redux application:
    * react-router
    * redux
    * lodash

# Testing

Uses specific packages for testing both react and redux:
    * chai
    * mocha
    * enzyme
    * jsdom

# Getting Started

```
git clone git@github.com:Yoyo2Code/ReduxStarter.git ReduxStarter
cd ReduxStarter
npm install
npm start
```
